IMAGENS NECESSÁRIAS PARA O SITE

Para completar o site, é necessário adicionar as seguintes imagens:

1. church-interior.jpg
   - Foto do interior da igreja com proporção 16:9
   - Recomendado: 1920x1080px
   - Esta imagem será usada como fundo da seção hero
   - Preferência por uma imagem levemente escurecida para melhor contraste com o texto

2. og-image.jpg
   - Imagem para compartilhamento em redes sociais
   - Tamanho recomendado: 1200x630px
   - Deve conter o logo da igreja e um fundo representativo

3. icon-192x192.png e icon-512x512.png
   - Ícones para PWA (Progressive Web App)
   - Podem ser versões redimensionadas do logo

NOTA: O logo.svg já está presente no diretório.
